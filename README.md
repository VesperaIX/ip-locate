<h1 align="center">IP-Locate</h1>

<br>

Simple IP Tracker Tool

# Instalation
`$ git clone https://github.com/VesperaIX/ip-locate && cd ip-locate`<br>
`$ pip3 install requests argparse`

# Usage
`$ python3 locate.py -i <ip_address>`
